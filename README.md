# Holiday Taxis Candidate Coding Test

Welcome to the Holiday Taxis candidate coding test!

Your task is to deploy this application to a single EC2 instance, simples.

## Constraints

- Any infrastructure created must be done via Terraform
- Configuration of the instance should be done via Ansible
- The instance should be:
- - Located in the `CandidateTest` VPC within the EU-West-1 region
- - Use the base Ubuntu Server 18.04 LTS AMI
- - Use the `CandidateTest` keypair

## General Information

**AWS Console:** https://htx-recruit.signin.aws.amazon.com/console

**Development VM:** sandbox.htxdev.com

Credentials for both will be provided.

The development VM has Ansible, Terraform and AWS CLI installed.

Feel free to install any other software as needed.

Ask questions; we're a collaborative team, we don't work in isolation.
